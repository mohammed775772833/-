### كيفية الاستخدام:
1. افتح محرر نصوص (مثل Notepad أو Visual Studio Code).
2. أنشئ ثلاث ملفات جديدة: `index.html`, `vision.html`, و`contact.html`.
3. انسخ الكود الخاص بكل صفحة في ملفها الخاص.
4. احفظ الملفات.
5. افتح `index.html` باستخدام متصفح الويب لديك، ويمكنك التنقل بين الصفحات.

### ملاحظات:
- يمكنك تعديل النصوص والألوان حسب الحاجة.
- تأكد من أن جميع الملفات موجودة في نفس المجلد لتعمل الروابط بشكل صحيح.

إذا كان لديك أي طلبات أو تعديلات إضافية، فلا تتردد في إخباري!
